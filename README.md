# Arithmetic-Math-Quiz
• This is my app that produces simple random arithmetic math questions.  
• The questions has addition, subtraction, multiplication and division.  
• Each game has 10 questions.  
• It has 3 separate levels of difficulty.  
• The minimum whole number should be 1 and maximum number should be 12.  
• Subtraction & Division should not generate negative numbers.  
• There is a score feature that will add up for each correct answer.  
• Each level will have an individual timer. (Level 0 = No Time Limit, Level 1 = 20 seconds, Level 2 = 10 seconds).  
• The graphics will be suitable for the target audience.  
• Will consist of sound effects to enhance more engagement for the audience.
